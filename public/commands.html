Herstart de bot via Discord zelf. Alleen server-admins.

Staff-rollen

Deze rollen mogen moderatie-commands en tickets beheren.

Collaborators

Geef iemand toegang tot dit dashboard via zijn/haar Discord ID (kan bot starten/stoppen, commands en modules beheren — maar niet verwijderen).

Logkanaal

Stuurt gebeurtenissen (zoals nieuwe tickets) naar een kanaal op je server.

Discord Webhook

Extra optie naast (of in plaats van) het logkanaal: stuur dezelfde start/stop/herstart-meldingen naar een webhook-URL. Handig als je al een webhook hebt draaien in een ander kanaal of systeem.

Maak je eigen webhook aan via Server-instellingen → Integraties → Webhooks in Discord. Deel deze link met niemand — wie 'm heeft, kan berichten in dat kanaal sturen.
